Layout

In this section we'll cover the layout of the Explore Screen. Do you remember that section? Here you have a screenshot of it:

On the left picture you can view the Explore screen as usual, while the picture on the right has more spacing between the blocks. I have modified that spacing to let you realise that this is a list with different layout items. Maybe if I show you this code you could understand it better.

As you can see, this is the structure of the list described above, it's an array of objects. Each item has an id field to use it as the key of that item, a layout field to select the layout that you want that item to be and another important field is photos where you specify the uri of the photos that will be shown.

Another field is customPhotoData, to give more info about each photo, this info will be passed to the detail screen.
